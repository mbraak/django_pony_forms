Installation
============

Install the package:

::

    $ pip install django_pony_forms

Add ``django_pony_forms`` to your installed apps in ``settings.py``.

::

    INSTALLED_APPS = (
        ..
        'django_pony_forms',
    )


Requirements
============

The package is tested with Django 2.2 - 3.1 and Python 3.6-3.8.
